1. Network 는 무엇이며 무엇을 목적으로 하는가?
	Net+work 망을 연결하여 서로 통신

2. Network 의 종류는 크게 ooo/ooo 로 나눌 수 있다
	LAN/WAN

3. LAN 의 특징에 대해서 서술하세요
	근거리 통신, 빠른 속도, 설치유지보수 간단, 보안 용이
	라우터, 스위치 등

4. WAN의 특징에 대해서 서술하세요
	장거리 통신(광역통신망), LAN & LAN 또는 다른 네트워크 연결, 광범위, 높은 비용, 관리 복잡
	전용회선, MPLS, VPN 등

5. InterNet 이란
	여러 종류의  통신망을 연결한 네트워크, 누구나 접근가능
	ex) 웹, 이메일, 파일전송 등

6. 프로토콜은 무엇인가?
	데이터 통신을 위한 규칙(컴퓨터나 장치끼리 통신하기 위해 지켜야하는 규칙) / 형식, 방법, 에러
	구성 요소 
		1. 구문(Syntax) : 데이터 형식, 구조 - 일관성 유지 
		2. 의미(Meaning) : 데이터 의믜, 목적 - 통신된 데이터 해석 및 응답
		3. 타이밍(Timing) : 데이터 전송 및 처리 - 전송, 응답시간, 재전송 ) 효율,안정 보장, 대기 시간 최소화(성능업)

7. 네트워킹 모델의 목적은 무엇인가?
	네트워크 구축 설계도, 효율적인 설계, 문제 탐색 용이, 이기종간 통신 정의 표준 프로토콜


8. TCP/IP 모델과 OSI 7 모델의 계층을 적고 각 계층별 역할을 적으세요
	OSI 7 모델
		1. 물리계층
		     장치 연결

		2. 데이터링크 계층
 		     운송 방식 결졍

		3. 네트워크 계층
		     경로 탐색
			
		4. 전송 계층		
		     데이터 전송(데이터 신뢰성 보장)
			
		5. 세션 계층
		     세션 관리, 연결 및 해제, 동기화
			
		6. 프레젠테이션 계층
		     데이터 표현 및 변환(암호화,복호화)
			
		7. 애플리케이션 계층
		     사용자와 상호작용하는 SW지원
		     			
	TCI/IP 모델
		1. 네트워크 인터페이스 계층(물리, 데이터링크)
		     물리적인 주소로 MAC을 사용한다

		2. 인터넷 계층
		     IP패킷 전송, 라우팅

		3. 전송 계층
		     통신 노드 연결(신뢰성), 데이터 전송

		4. 애플리케이션 계층(세션, 프레젠테이션, 애플리케이션)
		     TCP/UDP 기반의 응용 프로그램을 구현
