NETWORK IP : 네트워크 영역 식별
HOSP IP : 호스트간 식별

SUBNET MASK : 네트워크와 호스트 IP를 지정하기 위해 사용 / 32bit
1 : 네트워크 / 0 : 호스트

클래스풀 : IP주소를 규격화된 크기별(클래스)로 구분 시키는 방식
A 클래스 / 255.0.0.0 -> /8 -> 0.0.0.0~127.255.255.255
B 클래스 / 255.255.0.0 -> /16 -> 128.0.0.0 ~ 191.255.255.255
C 클래스 / 255.255.255.0 -> /24 -> 192.0.0.0 ~ 223.255.255.255
D 클래스 / 224.0.0.0 ~ 239.255.255.255
C 클래스 / 240.0.0.0 ~ 255.255.255.255

클래스리스란 : 규격을 따르지 않고 비트 단위별로 IP주소 범위를 가변적으로 구현

IPv4 통신방식 
Unicast - 트래픽 발송시 다수의 호스트에게 전달시 전달될 호스트 수만큼 복사하여 전달 > 1:1 통신
		호스트 수가 많으면 많은 트래픽 발생 

Broadcast - 트래픽 발송시 하나의 트래픽으로 모든 호스트에게 전달 > 1:n 통신
		모두에게 보냄으로 보안 취약

Multicast - Unicast + Broadcast 장점을 결합 > 1:n통신
